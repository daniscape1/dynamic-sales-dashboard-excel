# Dynamic-sales-dashboard-excel
This project is a dynamic Excel dashboard built to analyze Indian e-commerce sales and profit data in an interactive way.

## What this project covers

1.Dynamic dashboard with clean layout

2.Calculated fields for deeper metrics

3.Year-over-Year (YoY) analysis

4.Sparklines for quick trend tracking

4.Custom charts and formatted maps

6.Interactive filtering for easy exploration

## Objective

1.To convert raw sales data into meaningful insights that help understand:

2.Sales and profit trends

3.Performance changes over time

4.High- and low-performing periods

## Tools Used

### Microsoft Excel

1.Pivot Tables

2.Calculated Fields

3.Custom Charts & Maps

4.Sparklines

This is a portfolio project focused on analysis and visualization, not just data presentation.
